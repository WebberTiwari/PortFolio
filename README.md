```

├── public/              # Static assets (favicon, images, etc.)

├── components/          # Reusable UI components
├── styles/              # Global styles (TailwindCSS, CSS modules)

├── .gitignore

└── package.json         # Dependencies and scripts
```

## 🛠️ Tech Stack
- **Framework**: Next.js 13+
- **Styling**: TailwindCSS
- **Animation**: Framer Motion
